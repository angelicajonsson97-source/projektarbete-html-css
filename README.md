Tillgänglig och responsiv hemsida

# syfte
Detta projekt är en del av min kurs i HTML och CSS. Målet var att bygga en tillgänglig, responsiv och användarvänlig hemsida.

Designen är mobil-först med tydlig typografi, hög kontrast och intuitiv navigering. Layouten är byggd med flexbox och sticky footer.
